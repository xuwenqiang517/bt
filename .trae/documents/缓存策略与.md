## 优化计划

### 1. 缓存策略调整
- **`a_strategy_results.csv`**: 仅保存周期胜率100%的完整策略（7列）
- **`b_strategy_results.csv`**: 保存非100%胜率的策略（仅1列"配置"，用于去重）

### 2. 剪枝优化
遍历每个周期时：
- 如果某周期收益率 ≤ 0，立即跳过后续周期
- 标记为 `all_win = False`
- 该策略只保存到B缓存

### 3. 执行逻辑
```python
for s, e in self.date_arr:
    result = self.execute_one_strategy(strategy, s, e)
    if result.总收益率 <= 0:
        all_win = False
        break  # 剪枝
    results.append(result)

if all_win:
    保存到A缓存
else:
    保存到B缓存（仅配置列）
```

### 4. 预期效果
- 大幅减少无效策略的计算时间
- 减少B缓存文件大小（只存1列）
- 保持A缓存的完整性
